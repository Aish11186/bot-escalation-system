# Customer Support Intelligence System 
This is a plug-and-play ML middleware that predicts optimal human handoff points in chatbot conversations, reducing silent
churn in automated support systems for e-marketplaces and D2C startups.
It's trained on 400+ chatbot interaction logs and includes NLP features like sentiment trajectory, repetition and fallback detection. 
Deployed a regression model via a Flask API.
This system achieves 2–3s faster decision-making than LLM-based solutions, enabling cost reduction and real-time insights
